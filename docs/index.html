Welcome to the LDraw AIOI for Linux

this Installer is a highly useful piece of software that saves you the hassle of having to download different LDraw tools from different places and sites on linux. It installs the LDraw tools, library, Pov-ray, & wine, and is designed for ease of use. Just run the file with bash, select what you want to install, and you're away. (This installer needs bash. )
